
# Prosedur Penggunaan Aplikasi & Akun

Berikut tatacara menjalankan perintah sebelum menggunakan aplikasi agar pengalaman menggunakannya lebih baik.

1. Restore database
Restore terlebih dahulu database dengan file yang ada pada folder "SQL File"

2. Jalankan perintah scheduler
Perintah ini digunakan agar pesanan yang sudah melebihi batas waktu yang ditentukan akan berubah statusnya secara otomatis

php artisan schedule:run

3. Jalankan perintah PHP artisan serve
 Aplikasi siap digunakan
